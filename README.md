# larptrack

literally just polytrack but i was bored and wanted to add stuff without actually coding it. w google antigravity.

# how to use
download it, if you really want to use electron to compile $ROOT/downloadsdir/larptrack/files to an exe if your on windows, if youre not just use python to set up a local http server. if you dont know how to do that, you:

1. install python
2. cd to the $ROOT/downloads/larptrack/files directory
3. run `python -m http.server 8000`
4. go to http://localhost:8000
